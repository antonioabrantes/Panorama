                                                              PARECER
          O presente pedido refere-se a um codificador para decodificar um fluxo contínuo de
dados (12) compreendendo um bloco de configuração (28) 5/17 e uma sequência de estruturas
(20) respectivamente representando os períodos de tempo consecutivos de um conteúdo de
áudio (10),          caracterizado          pela sequência             de estruturas (20) ser   uma composição
de N sequências de elementos da estrutura (22) com cada elemento da estrutura (22) sendo de
um respectivo tipo de uma pluralidade de tipos de elemento de modo que cada estrutura (20)
compreende um elemento da estrutura (22) fora da N sequências de elementos da estrutura
( 22) , respectivamente, e para cada sequência de elementos da estrutura (22), os elementos da
estrutura (22) são do tipo de elemento igual com relação entre si.
